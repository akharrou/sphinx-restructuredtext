################################################################################
RST ➤ Images
################################################################################

**********************************************************************
Synopsis
**********************************************************************

Images are supported with the ``images`` directive.

**********************************************************************
Examples
**********************************************************************

Basic
============================================================

In
    ::

        .. image:: _assets/lion.jpg

Out

        .. image:: _assets/lion.jpg

Custom output
============================================================

In
    ::

        .. image:: _assets/lion.jpg
            :height: 100
            :width: 200
            :scale: 50
            :alt: alternate text

Out

        .. image:: _assets/lion.jpg
            :height: 100
            :width: 200
            :scale: 50
            :alt: alternate text
