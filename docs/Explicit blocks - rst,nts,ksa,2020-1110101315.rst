RST ➤ Explicit markup blocks
################################################################################

**********************************************************************
Synopsis
**********************************************************************

Explicit blocks are text blocks:

- which begin with ".. "
- whose subsequent lines (if any) are indented
- which end at the first unindented line

They are used for:

- footnotes
- citations
- hyperlink targets
- directives
- substitution definitions
- comments
