################################################################################
RST ➤ Formatting
################################################################################

**********************************************************************
Synopsis
**********************************************************************

You have the following facilities for formatting:

- emphasis (italics), with ``*text*`` syntax
- strong emphasis (boldface), with ``**text**`` syntax

**********************************************************************
Examples
**********************************************************************

In

.. code-block:: rst

    some *text* is italic

    some other **text** is bold

Out

some *text* is italic

some other **text** is bold
